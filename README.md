# (게임 기획서)

---

## 1. 스토리 요약
- 기억을 잃은 소녀가 말 없는 늑대와 함께 피리를 회복하기 위한 여정을 떠남.  
- 소녀는 피리를 사용해 **온화한 공격(정화)** 또는 **난폭한 공격(오염)** 을 선택할 수 있음.  
- 여정을 통해 자신의 과거 상처와 폭력의 기억을 마주하고, 성장과 치유를 경험함.  
- 최종적으로 보스가 **자기 자신**임을 깨닫고, 스스로를 용서하며 현실로 돌아옴.  

---

## 2. 스토리 흐름
### 1막: 여정의 시작 (기억 상실)
- 기억을 잃은 소녀, 늑대와 동행  
- 피리를 통해 기본 전투/상호작용 학습  
- 몬스터를 사냥하며 점차 음계를 수집  

### 2막: 타락과 희생 (되물림)
- 난폭한 공격(날카로운 연주) 위주 → 강해지지만 오염 증가  
- 늑대는 온화한 방법을 조언하지만 점차 희생  
- 거울의 성에서 늑대가 소녀를 지키고 쓰러짐  

### 3막: 회귀와 정화 (내면화)
- 1스테이지로 회귀, 늑대 없음  
- 늑대가 지켜주었던 **오염도 UI**가 **HP UI**로 변경, 활성화  
- Echo Guard(늑대 울음 모방) 습득  
- 온화한 공격이 강화되며 스스로 성장  
- 최종 보스 = 자기 자신 → 치유와 화해  

---

## 3. 주요 시스템

### 피리 시스템
- **분노의 악장 (F 단타)**: 즉사 공격, 오염 증가  
- **평화의 악장 (F 홀드)**: 정화, 오염 감소  

### 오염도 시스템
- 오염도가 높을수록 공격력 증가  
- 대신 자가 데미지 발생  

### Echo Guard (감정 방어)
- Shift 키로 발동  
- 범위, 지속 시간, 효과가 성장형  

### 회귀 시스템
- 늑대 희생 후 1스테이지로 회귀  
- 회귀 후 장애물 감소 → 성장 연출  

---

## 4. 스테이지별 구조

| 스테이지 | 테마 | 기믹 | 클리어 조건 |
|----------|------|------|-------------|
| 1. 무언의 숲 (무관심) | 무음, 안개 숲 | 피리 사용 불가, 감정 표현만 가능 | 정화 씨앗 3개 수집 후 문 개방 |
| 2. 메마른 정원 (결핍) | 빛나는 숲 | 어두운 곳 데미지, 씨앗 심기 | 빛나는 꽃 3개 수집 후 포탈 생성 |
| 3. 우거진 숲 (폭력) | 번개 스턴 | 다수 몬스터, 날카로운 공격 유도 | 중간 보스 처치 |
| 4. 거울의 성 (되물림) | 거울 맵, 보스 소녀 | 맵 변형, 자기혐오 대사 출력 | 최종 보스 처치 |

---

## 5. 개발 포인트
- 감정 애니메이션(행복/슬픔) 활용  
- 배경·효과음이 회귀 이후 변화 → 성장 반영  
- 오염도 UI → 초반엔 비활성화, 회귀 후 활성화  
- 컷신과 대사로 **폭력의 되물림 → 자기 치유** 테마 강화  

---

## 6. 엔딩
- 최종 전투 후, 오염도 수치에 따라 엔딩 분기  
- 늑대와의 재회 → 어른이 된 소녀가 자신의 딸을 품에 안는 장면으로 마무리  
- **최종 문구**  
  > “보호받는 아이에서, 누군가의 늑대가 되기까지.”

## Git 환경 준비
1. **Git Bash 설치 여부 확인**  
   - Windows 환경에서 Git Bash가 설치되어 있는지 확인합니다.  

2. **항상 개발 시작하기전에 git pull 필수****
   - 깃 배쉬 -> git clone (주소)

3. **초기화하는 법**
    - powershell창
    - cd (해당경로)
    - git fetch origin
    - git reset --hard origin/main
