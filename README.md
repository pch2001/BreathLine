## Git 환경 준비
1. **Git Bash 설치 여부 확인**  
   - Windows 환경에서 Git Bash가 설치되어 있는지 확인합니다.  

2. **항상 개발 시작하기전에 git pull 필수****
   - 깃 배쉬 -> git clone (주소)

3. **초기화하는 법**
    - powershell창
    - cd (해당경로)
    - git fetch origin
    - git reset --hard origin/main
